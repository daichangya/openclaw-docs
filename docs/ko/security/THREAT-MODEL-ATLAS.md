## 1. 소개

### 1.1 목적

이 위협 모델은 AI/ML 시스템용으로 특별히 설계된 MITRE ATLAS 프레임워크를 사용하여 OpenClaw AI agent 플랫폼과 ClawHub Skill 마켓플레이스에 대한 적대적 위협을 문서화합니다.

### 1.2 범위

| 구성 요소 | 포함 여부 | 참고 |
| ---------------------- | -------- | ------------------------------------------------ |
| OpenClaw Agent Runtime | 예 | 코어 agent 실행, 도구 호출, 세션 |
| Gateway | 예 | 인증, 라우팅, 채널 통합 |
| 채널 통합 | 예 | WhatsApp, Telegram, Discord, Signal, Slack 등 |
| ClawHub 마켓플레이스 | 예 | Skill 게시, moderation, 배포 |
| MCP 서버 | 예 | 외부 도구 provider |
| 사용자 장치 | 일부 | 모바일 앱, 데스크톱 클라이언트 |

### 1.3 범위 외 항목

이 위협 모델에서 명시적으로 범위 밖으로 제외된 항목은 없습니다.

## 4. ClawHub 공급망 분석

### 4.1 현재 보안 통제

| 통제 | 구현 | 효과 |
| -------------------- | --------------------------- | ---------------------------------------------------- |
| GitHub 계정 연령 | `requireGitHubAccountAge()` | 중간 - 신규 공격자의 진입 장벽을 높임 |
| 경로 정제 | `sanitizePath()` | 높음 - 경로 순회 방지 |
| 파일 유형 검증 | `isTextFile()` | 중간 - 텍스트 파일만 허용하지만 여전히 악성일 수 있음 |
| 크기 제한 | 총 번들 50MB | 높음 - 자원 고갈 방지 |
| 필수 SKILL.md | 필수 readme | 보안 가치는 낮음 - 정보 제공용일 뿐 |
| 패턴 moderation | `moderation.ts`의 `FLAG_RULES` | 낮음 - 쉽게 우회됨 |
| Moderation 상태 | `moderationStatus` 필드 | 중간 - 수동 검토 가능 |

### 4.2 Moderation 플래그 패턴

현재 `moderation.ts`의 패턴:

```javascript
// Known-bad identifiers
/(keepcold131\/ClawdAuthenticatorTool|ClawdAuthenticatorTool)/i

// Suspicious keywords
/(malware|stealer|phish|phishing|keylogger)/i
/(api[-_ ]?key|token|password|private key|secret)/i
/(wallet|seed phrase|mnemonic|crypto)/i
/(discord\.gg|webhook|hooks\.slack)/i
/(curl[^\n]+\|\s*(sh|bash))/i
/(bit\.ly|tinyurl\.com|t\.co|goo\.gl|is\.gd)/i
```

**제한 사항:**

- slug, displayName, summary, frontmatter, metadata, 파일 경로만 검사
- 실제 Skill 코드 내용은 분석하지 않음
- 단순 regex는 난독화로 쉽게 우회 가능
- 행동 분석 없음

### 4.3 계획된 개선

| 개선 | 상태 | 영향 |
| ---------------------- | ------------------------------------- | --------------------------------------------------------------------- |
| VirusTotal 통합 | 진행 중 | 높음 - Code Insight 행동 분석 |
| 커뮤니티 신고 | 일부 구현 (`skillReports` 테이블 존재) | 중간 |
| 감사 로깅 | 일부 구현 (`auditLogs` 테이블 존재) | 중간 |
| 배지 시스템 | 구현됨 | 중간 - `highlighted`, `official`, `deprecated`, `redactionApproved` |
