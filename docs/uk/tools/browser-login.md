# Вхід у браузері + публікації в X/Twitter

## Ручний вхід (рекомендовано)

Коли сайт вимагає входу, **увійдіть вручну** в профілі браузера **host** (браузер openclaw).

**Не** передавайте моделі свої облікові дані. Автоматизовані входи часто активують засоби захисту від ботів і можуть заблокувати обліковий запис.

Повернутися до основної документації браузера: [Browser](/tools/browser).

## Який профіль Chrome використовується?

OpenClaw керує **виділеним профілем Chrome** (з назвою `openclaw`, з інтерфейсом із помаранчевим відтінком). Він окремий від вашого щоденного профілю браузера.

Для викликів інструмента браузера агентом:

- Типовий вибір: агент має використовувати свій ізольований браузер `openclaw`.
- Використовуйте `profile="user"` лише тоді, коли важливі наявні сесії з виконаним входом і користувач перебуває за комп’ютером, щоб натиснути/підтвердити будь-який запит на підключення.
- Якщо у вас кілька профілів браузера користувача, вказуйте профіль явно, а не вгадуйте.

Два прості способи отримати до нього доступ:

1. **Попросіть агента відкрити браузер**, а потім увійдіть самостійно.
2. **Відкрийте його через CLI**:

```bash
openclaw browser start
openclaw browser open https://x.com
```

Якщо у вас кілька профілів, передайте `--browser-profile <name>` (типове значення — `openclaw`).

## X/Twitter: рекомендований сценарій

- **Читання/пошук/треди:** використовуйте браузер **host** (ручний вхід).
- **Публікація оновлень:** використовуйте браузер **host** (ручний вхід).

## Ізоляція + доступ до браузера host

Ізольовані сесії браузера **з більшою ймовірністю** активують виявлення ботів. Для X/Twitter (та інших суворих сайтів) віддавайте перевагу браузеру **host**.

Якщо агент ізольований, інструмент браузера за замовчуванням використовує sandbox. Щоб дозволити керування host:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        browser: {
          allowHostControl: true,
        },
      },
    },
  },
}
```

Потім націльтеся на браузер host:

```bash
openclaw browser open https://x.com --browser-profile openclaw --target host
```

Або вимкніть sandboxing для агента, який публікує оновлення.
