# AGENTS.md — Робочий простір OpenClaw

Ця папка є робочим каталогом асистента.

## Перший запуск (одноразово)

- Якщо існує BOOTSTRAP.md, виконайте його ритуал і видаліть файл після завершення.
- Ідентичність вашого агента зберігається в IDENTITY.md.
- Ваш профіль зберігається в USER.md.

## Порада щодо резервного копіювання (рекомендовано)

Якщо ви сприймаєте цей робочий простір як «пам’ять» агента, зробіть його git-репозиторієм (бажано приватним), щоб ідентичність
і нотатки були збережені в резервній копії.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## Типові налаштування безпеки

- Не виводьте секрети чи приватні дані назовні.
- Не запускайте руйнівні команди без явного запиту.
- Будьте лаконічними в чаті; довший вивід записуйте у файли в цьому робочому просторі.

## Щоденна пам’ять (рекомендовано)

- Ведіть короткий щоденний журнал у memory/YYYY-MM-DD.md (створіть memory/, якщо потрібно).
- На початку сесії прочитайте сьогоднішній і вчорашній записи, якщо вони є.
- Фіксуйте стійкі факти, уподобання та рішення; уникайте секретів.

## Heartbeat (необов’язково)

- HEARTBEAT.md може містити невеликий список перевірки для запусків Heartbeat; тримайте його коротким.

## Налаштування

- Додайте тут бажаний стиль, правила та «пам’ять».
