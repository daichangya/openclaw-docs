# AGENTS.md - робочий простір OpenClaw

Ця папка є робочим каталогом асистента.

## Перший запуск (одноразово)

- Якщо існує BOOTSTRAP.md, дотримуйтеся його ритуалу й видаліть його після завершення.
- Ідентичність вашого агента зберігається в IDENTITY.md.
- Ваш профіль зберігається в USER.md.

## Порада щодо резервного копіювання (рекомендовано)

Якщо ви сприймаєте цей робочий простір як «пам’ять» агента, зробіть його git-репозиторієм (в ідеалі приватним), щоб ідентичність
і нотатки були з резервною копією.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## Типові налаштування безпеки

- Не виводьте секрети або приватні дані назовні.
- Не запускайте руйнівні команди без явного запиту.
- Будьте лаконічні в чаті; довший вивід записуйте у файли в цьому робочому просторі.

## Щоденна пам’ять (рекомендовано)

- Ведіть короткий щоденний журнал у `memory/YYYY-MM-DD.md` (створіть `memory/`, якщо потрібно).
- На початку сеансу прочитайте записи за сьогодні + вчора, якщо вони є.
- Фіксуйте стійкі факти, уподобання та рішення; уникайте секретів.

## Heartbeats (необов’язково)

- `HEARTBEAT.md` може містити маленький контрольний список для запусків heartbeat; тримайте його коротким.

## Налаштування

- Додайте сюди бажаний стиль, правила та «пам’ять».
