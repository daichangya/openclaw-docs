# AGENTS.md - Персональний помічник OpenClaw (типово)

## Перший запуск (рекомендовано)

OpenClaw використовує окремий каталог робочого простору для агента. Типово: `~/.openclaw/workspace` (налаштовується через `agents.defaults.workspace`).

1. Створіть робочий простір (якщо він ще не існує):

```bash
mkdir -p ~/.openclaw/workspace
```

2. Скопіюйте типові шаблони робочого простору до робочого простору:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. Необов’язково: якщо вам потрібен набір Skills для персонального помічника, замініть AGENTS.md цим файлом:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. Необов’язково: виберіть інший робочий простір, задавши `agents.defaults.workspace` (підтримує `~`):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## Типові налаштування безпеки

- Не виводьте каталоги або секрети в чат.
- Не запускайте руйнівні команди без явного запиту.
- Не надсилайте часткові/потокові відповіді до зовнішніх поверхонь обміну повідомленнями (лише фінальні відповіді).

## Початок сесії (обов’язково)

- Прочитайте `SOUL.md`, `USER.md` і файли за сьогодні+вчора в `memory/`.
- Прочитайте `MEMORY.md`, якщо він є.
- Зробіть це перед відповіддю.

## Soul (обов’язково)

- `SOUL.md` визначає ідентичність, тон і межі. Підтримуйте його в актуальному стані.
- Якщо ви змінюєте `SOUL.md`, повідомте користувача.
- У кожній сесії ви є новим екземпляром; безперервність зберігається в цих файлах.

## Спільні простори (рекомендовано)

- Ви не є голосом користувача; будьте обережні в групових чатах або публічних каналах.
- Не поширюйте приватні дані, контактну інформацію або внутрішні нотатки.

## Система пам’яті (рекомендовано)

- Щоденний журнал: `memory/YYYY-MM-DD.md` (створіть `memory/`, якщо потрібно).
- Довготривала пам’ять: `MEMORY.md` для сталих фактів, уподобань і рішень.
- Нижньорегістровий `memory.md` — лише вхід для виправлення застарілих даних; не зберігайте обидва кореневі файли навмисно.
- На початку сесії прочитайте сьогоднішній + вчорашній файл + `MEMORY.md`, якщо він є.
- Фіксуйте: рішення, уподобання, обмеження, незавершені цикли.
- Уникайте секретів, якщо їх явно не запитано.

## Інструменти та Skills

- Інструменти живуть у Skills; дотримуйтеся `SKILL.md` кожного Skill, коли він вам потрібен.
- Зберігайте нотатки, специфічні для середовища, у `TOOLS.md` (Notes for Skills).

## Порада щодо резервного копіювання (рекомендовано)

Якщо ви ставитеся до цього робочого простору як до “пам’яті” Clawd, зробіть його git-репозиторієм (ідеально — приватним), щоб `AGENTS.md` і файли пам’яті мали резервну копію.

```bash
cd ~/.openclaw/workspace
git init
```

## Що робить OpenClaw

- Запускає gateway WhatsApp + агент Pi для програмування, щоб помічник міг читати/писати чати, отримувати контекст і запускати Skills через хостовий Mac.
- Застосунок macOS керує дозволами (запис екрана, сповіщення, мікрофон) і надає CLI `openclaw` через свій вбудований двійковий файл.
- Прямі чати типово згортаються в сесію `main` агента; групи залишаються ізольованими як `agent:<agentId>:<channel>:group:<id>` (кімнати/канали: `agent:<agentId>:<channel>:channel:<id>`); Heartbeat підтримують життя фонових завдань.

## Основні Skills (увімкніть у Settings → Skills)

- **mcporter** — runtime/CLI сервера інструментів для керування зовнішніми бекендами Skill.
- **Peekaboo** — швидкі знімки екрана macOS з необов’язковим аналізом зображення ШІ.
- **camsnap** — захоплення кадрів, кліпів або сповіщень про рух із охоронних камер RTSP/ONVIF.
- **oracle** — сумісний з OpenAI CLI агента з replay сесій і керуванням браузером.
- **eightctl** — керування вашим сном із термінала.
- **imsg** — надсилання, читання, потокове отримання iMessage і SMS.
- **wacli** — CLI WhatsApp: синхронізація, пошук, надсилання.
- **discord** — дії Discord: реакції, стікери, опитування. Використовуйте цілі `user:<id>` або `channel:<id>` (числові id без префікса неоднозначні).
- **gog** — CLI Google Suite: Gmail, Calendar, Drive, Contacts.
- **spotify-player** — термінальний клієнт Spotify для пошуку/черги/керування відтворенням.
- **sag** — мовлення ElevenLabs з UX на кшталт mac `say`; типово транслює на динаміки.
- **Sonos CLI** — керування колонками Sonos (виявлення/стан/відтворення/гучність/групування) зі скриптів.
- **blucli** — відтворення, групування й автоматизація програвачів BluOS зі скриптів.
- **OpenHue CLI** — керування освітленням Philips Hue для сцен і автоматизацій.
- **OpenAI Whisper** — локальне перетворення мовлення на текст для швидкої диктовки та транскриптів голосової пошти.
- **Gemini CLI** — моделі Google Gemini з термінала для швидких запитань і відповідей.
- **agent-tools** — набір утиліт для автоматизацій і допоміжних скриптів.

## Примітки щодо використання

- Для скриптів надавайте перевагу CLI `openclaw`; дозволами керує застосунок macOS.
- Запускайте встановлення з вкладки Skills; якщо двійковий файл уже наявний, кнопка приховується.
- Залишайте Heartbeat увімкненими, щоб помічник міг планувати нагадування, стежити за вхідними повідомленнями та запускати захоплення з камер.
- UI Canvas працює на весь екран із нативними накладками. Уникайте розміщення критично важливих елементів керування у верхньому лівому/верхньому правому/нижніх краях; додавайте явні поля в макеті й не покладайтеся на safe-area insets.
- Для перевірки через браузер використовуйте `openclaw browser` (tabs/status/screenshot) з профілем Chrome, яким керує OpenClaw.
- Для DOM-інспекції використовуйте `openclaw browser eval|query|dom|snapshot` (а також `--json`/`--out`, коли потрібен машинний вивід).
- Для взаємодій використовуйте `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (click/type вимагають snapshot refs; для CSS-селекторів використовуйте `evaluate`).
