Індикатори набору тексту надсилаються в чат-канал, доки активний запуск. Використовуйте
`agents.defaults.typingMode`, щоб керувати **коли** починається набір тексту, а `typingIntervalSeconds`
— щоб керувати **як часто** він оновлюється.

## Значення за замовчуванням

Коли `agents.defaults.typingMode` **не задано**, OpenClaw зберігає застарілу поведінку:

- **Прямі чати**: набір тексту починається одразу, щойно починається цикл моделі.
- **Групові чати зі згадкою**: набір тексту починається одразу.
- **Групові чати без згадки**: набір тексту починається лише тоді, коли починає потоково надходити текст повідомлення.
- **Запуски Heartbeat**: набір тексту починається, коли починається запуск Heartbeat, якщо
  визначена ціль Heartbeat — це чат, що підтримує набір тексту, і якщо набір тексту не вимкнено.

## Режими

Установіть `agents.defaults.typingMode` в одне з таких значень:

- `never` — індикатор набору тексту ніколи не показується.
- `instant` — почати набір тексту **щойно починається цикл моделі**, навіть якщо запуск
  пізніше поверне лише тихий токен відповіді.
- `thinking` — почати набір тексту на **першій дельті міркування** (потребує
  `reasoningLevel: "stream"` для цього запуску).
- `message` — почати набір тексту на **першій нетихій текстовій дельті** (ігнорує
  тихий токен `NO_REPLY`).

Порядок “наскільки рано це спрацьовує”:
`never` → `message` → `thinking` → `instant`

## Конфігурація

```json5
{
  agent: {
    typingMode: "thinking",
    typingIntervalSeconds: 6,
  },
}
```

Ви можете перевизначити режим або частоту для кожної сесії:

```json5
{
  session: {
    typingMode: "message",
    typingIntervalSeconds: 4,
  },
}
```

## Примітки

- Режим `message` не показуватиме набір тексту для лише тихих відповідей, коли весь
  вміст є точним тихим токеном (наприклад, `NO_REPLY` / `no_reply`,
  зіставлення без урахування регістру).
- `thinking` спрацьовує лише якщо запуск потоково передає міркування (`reasoningLevel: "stream"`).
  Якщо модель не видає дельти міркування, набір тексту не почнеться.
- Набір тексту Heartbeat є сигналом активності для визначеної цілі доставки. Він
  починається на старті запуску Heartbeat замість дотримання таймінгу потоків `message` або `thinking`.
  Установіть `typingMode: "never"`, щоб вимкнути його.
- Heartbeat не показує набір тексту, коли `target: "none"`, коли ціль неможливо
  визначити, коли для Heartbeat вимкнено доставку в чат або коли
  канал не підтримує набір тексту.
- `typingIntervalSeconds` керує **частотою оновлення**, а не часом початку.
  Значення за замовчуванням — 6 секунд.
