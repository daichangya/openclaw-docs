# Ресурси i18n документації OpenClaw

У цій папці зберігається конфігурація перекладу для репозиторію вихідної документації.

Згенеровані сторінки локалей і актуальна пам’ять перекладу локалей тепер розміщені в репозиторії публікації (`openclaw/docs`, локальна сусідня копія `~/Projects/openclaw-docs`).

## Файли

- `glossary.<lang>.json` — зіставлення бажаних термінів (використовуються в підказках промпта).
- `<lang>.tm.jsonl` — пам’ять перекладу (кеш), прив’язана до workflow + model + хеша тексту. У цьому репозиторії файли TM локалей генеруються на вимогу.

## Формат глосарія

`glossary.<lang>.json` — це масив записів:

```json
{
  "source": "troubleshooting",
  "target": "故障排除",
  "ignore_case": true,
  "whole_word": false
}
```

Поля:

- `source`: англійська (або вихідна) фраза, якій слід надавати перевагу.
- `target`: бажаний результат перекладу.

## Примітки

- Записи глосарія передаються моделі як **підказки промпта** (без детермінованих переписувань).
- `scripts/docs-i18n` і надалі відповідає за генерацію перекладів.
- Репозиторій джерела синхронізує англійську документацію з репозиторієм публікації; генерація локалей запускається там для кожної локалі під час push, за розкладом і під час release dispatch.
