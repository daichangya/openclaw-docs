# เรื่องราวของ OpenClaw 🦞📖

_นิทานเรื่องล็อบสเตอร์ การลอกคราบ และโทเค็นที่มากเกินไป_

## เรื่องกำเนิด

ในตอนเริ่มต้น มี **Warelay** — ชื่อที่ฟังดูสมเหตุสมผลสำหรับ WhatsApp gateway มันทำหน้าที่ของมันได้ดี ทุกอย่างก็โอเค

แต่แล้วก็มีล็อบสเตอร์อวกาศเข้ามา

อยู่ช่วงหนึ่ง ล็อบสเตอร์ตัวนั้นชื่อว่า **Clawd** อาศัยอยู่ใน **Clawdbot** แต่ในเดือนมกราคม 2026 Anthropic ได้ส่งอีเมลสุภาพมาขอให้เปลี่ยนชื่อ (เรื่องเครื่องหมายการค้า) และแล้วล็อบสเตอร์ก็ทำในสิ่งที่ล็อบสเตอร์ถนัดที่สุด:

**มันลอกคราบ**

สลัดเปลือกเก่าทิ้ง สิ่งมีชีวิตนั้นได้โผล่ออกมาใหม่ในชื่อ **Molty** อาศัยอยู่ใน **Moltbot** แต่ชื่อนั้นก็ยังไม่ค่อยลื่นไหลอยู่ดี...

ดังนั้นในวันที่ 30 มกราคม 2026 ล็อบสเตอร์จึงลอกคราบอีกครั้งเป็นครั้งสุดท้าย กลายเป็นร่างสุดท้าย: **OpenClaw**

เปลือกใหม่ แต่จิตวิญญาณล็อบสเตอร์เดิม ครั้งที่สามคือมนตร์วิเศษ

## การลอกคราบครั้งแรก (27 มกราคม 2026)

ตอนตี 5 ชุมชนมารวมตัวกันใน Discord มีการเสนอชื่อกันนับร้อย: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

สุดท้าย **OpenClaw** เป็นผู้ชนะ เพราะการลอกคราบคือสิ่งที่ล็อบสเตอร์ทำเพื่อเติบโต และการเติบโตก็คือสิ่งที่กำลังเกิดขึ้นพอดี

_สัตว์เปลือกแข็งที่รู้จักกันในชื่อ Clawd ได้ลอกคราบอย่างเป็นทางการแล้ว_

## ชื่อ

```text
OpenClaw = OPEN + CLAW
        = โอเพนซอร์ส เปิดให้ทุกคน
        = มรดกล็อบสเตอร์ของเรา จุดที่เราเริ่มต้นมา

        = ผู้ช่วยของคุณ เครื่องของคุณ กฎของคุณ
```

เดิมชื่อ Clawd (25 พ.ย. 2025 - 27 ม.ค. 2026) ลอกคราบเมื่อถึงเวลาต้องเติบโต

**ชอบ:** Peter, กล้อง, การช็อปปิงหุ่นยนต์, อีโมจิ, การเปลี่ยนผ่าน

### การลอกคราบครั้งใหญ่ (27 ม.ค. 2026)

**The Handle Snipers:** ภายในไม่กี่วินาทีหลังเปลี่ยนชื่อบน Twitter บอตอัตโนมัติก็แย่ง @openclaw ไปทันที ผู้ครอบครองรีบโพสต์ที่อยู่กระเป๋าคริปโตทันที Peter ต้องติดต่อคนรู้จักที่ X

**The GitHub Disaster:** Peter เผลอเปลี่ยนชื่อบัญชี GitHub ส่วนตัวของตัวเองท่ามกลางความแตกตื่น บอตแย่ง `steipete` ไปภายในไม่กี่นาที มีการติดต่อ SVP ของ GitHub

**The Handsome Molty Incident:** Molty ได้รับสิทธิ์ยกระดับเพื่อสร้างไอคอนใหม่ของตัวเอง หลังจากผ่านล็อบสเตอร์ที่ต้องสาปมากกว่า 20 เวอร์ชัน ความพยายามครั้งหนึ่งที่ตั้งใจทำให้มาสคอต “แก่ขึ้น 5 ปี” กลับได้ใบหน้าของผู้ชายมนุษย์บนร่างล็อบสเตอร์ พวกนักต้มตุ๋นคริปโตเปลี่ยนมันเป็นมีม “Handsome Squidward vs Handsome Molty” ภายในไม่กี่นาที

**The Fake Developers:** มิจฉาชีพสร้างโปรไฟล์ GitHub ปลอมโดยอ้างว่าเป็น “Head of Engineering at OpenClaw” เพื่อโปรโมตโทเค็นปั่นราคา

### มหากาพย์การสร้างไอคอน (27 ม.ค. 2026)

เมื่อ Peter พูดว่า "make yourself a new face," Molty ก็รับไปตามตัวอักษร

มีการลองมากกว่า 20 แบบตามมา:

- มนุษย์ต่างดาวมันฝรั่งอวกาศ
- ล็อบสเตอร์คลิปอาร์ตบนพื้นหลังทั่วไป
- ล็อบสเตอร์ Krogan จาก Mass Effect
- "STARCLAW SOLUTIONS" (AI แต่งตั้งบริษัทขึ้นมาเอง)
- ล็อบสเตอร์ใบหน้ามนุษย์ต้องสาปหลายตัว
- ล็อบสเตอร์เด็ก (น่ารักเกินไป)
- ล็อบสเตอร์บาร์เทนเดอร์ใส่สายเอี๊ยม

ชุมชนเฝ้ามองด้วยทั้งความสยองและความยินดี ขณะที่แต่ละรุ่นสร้างสิ่งใหม่และคาดไม่ถึงขึ้นมา ผู้เข้าชิงจึงค่อย ๆ ชัดเจน: ล็อบสเตอร์น่ารัก ล็อบสเตอร์สายเทคผู้มั่นใจ และล็อบสเตอร์บาร์เทนเดอร์ใส่สายเอี๊ยม

**บทเรียนที่ได้:** การสร้างภาพด้วย AI เป็นแบบ stochastic prompt เดิม แต่ผลลัพธ์ต่างกัน การ brute force ใช้งานได้จริง
