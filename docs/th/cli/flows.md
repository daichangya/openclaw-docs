# `openclaw tasks flow`

คำสั่ง Flow เป็นคำสั่งย่อยของ `openclaw tasks` ไม่ใช่คำสั่ง `flows` แบบแยกต่างหาก

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```
