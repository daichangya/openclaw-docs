# TOOLS.md - บันทึกในเครื่อง

Skills กำหนดว่าเครื่องมือ _ทำงานอย่างไร_ ไฟล์นี้มีไว้สำหรับรายละเอียด _เฉพาะของคุณ_ — สิ่งที่เป็นเอกลักษณ์ของสภาพแวดล้อมคุณ

## สิ่งที่ควรใส่ไว้ที่นี่

สิ่งอย่างเช่น:

- ชื่อและตำแหน่งของกล้อง
- โฮสต์และ aliases ของ SSH
- เสียงที่ชอบสำหรับ TTS
- ชื่อของลำโพง/ห้อง
- ชื่อเล่นของอุปกรณ์
- ทุกอย่างที่เฉพาะกับสภาพแวดล้อม

## ตัวอย่าง

```markdown
### กล้อง

- living-room → พื้นที่หลัก มุมกว้าง 180°
- front-door → ทางเข้า เปิดด้วยการตรวจจับการเคลื่อนไหว

### SSH

- home-server → 192.168.1.100, ผู้ใช้: admin

### TTS

- เสียงที่ชอบ: "Nova" (อบอุ่น ติดสำเนียงบริติชนิดๆ)
- ลำโพงเริ่มต้น: Kitchen HomePod
```

## ทำไมต้องแยกต่างหาก?

Skills เป็นสิ่งที่ใช้ร่วมกัน ส่วนการตั้งค่าของคุณเป็นของคุณเอง การแยกสองอย่างนี้ออกจากกันหมายความว่าคุณอัปเดต skills ได้โดยไม่สูญเสียบันทึกของคุณ และแชร์ skills ได้โดยไม่ทำให้โครงสร้างพื้นฐานของคุณรั่วไหล
