# TOOLS.md - บันทึกเครื่องมือของผู้ใช้ (แก้ไขได้)

ไฟล์นี้มีไว้สำหรับ _บันทึกของคุณเอง_ เกี่ยวกับเครื่องมือภายนอกและข้อตกลงการใช้งาน
มันไม่ได้กำหนดว่าเครื่องมือใดมีอยู่บ้าง; OpenClaw มี built-in tool ให้ภายในอยู่แล้ว

## ตัวอย่าง

### imsg

- ส่ง iMessage/SMS: อธิบายว่าจะส่งถึงใคร/ส่งอะไร และยืนยันก่อนส่ง
- ควรใช้ข้อความสั้น ๆ; หลีกเลี่ยงการส่ง secret

### sag

- แปลงข้อความเป็นเสียงพูด: ระบุเสียง ลำโพง/ห้องเป้าหมาย และต้องการสตรีมหรือไม่

เพิ่มสิ่งใดก็ได้ที่คุณอยากให้ผู้ช่วยรู้เกี่ยวกับ toolchain ภายในเครื่องของคุณ
