## เคล็ดลับเรื่องการสำรองข้อมูล (แนะนำ)

หากคุณถือว่า workspace นี้เป็น "memory" ของเอเจนต์ ให้ทำมันเป็น git repo (ควรเป็น private) เพื่อให้ตัวตน
และบันทึกต่าง ๆ ถูกสำรองไว้

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## ค่าเริ่มต้นด้านความปลอดภัย

- อย่านำความลับหรือข้อมูลส่วนตัวออกไป
- อย่ารันคำสั่งที่ทำลายข้อมูล เว้นแต่จะถูกร้องขออย่างชัดเจน
- ให้กระชับในแชต; หากเป็นเอาต์พุตที่ยาวกว่า ให้เขียนลงไฟล์ใน workspace นี้

## memory รายวัน (แนะนำ)

- เก็บบันทึกรายวันสั้น ๆ ไว้ที่ `memory/YYYY-MM-DD.md` (สร้าง `memory/` หากจำเป็น)
- เมื่อเริ่มเซสชัน ให้อ่านวันนี้ + เมื่อวานหากมี
- บันทึกข้อเท็จจริง ความชอบ และการตัดสินใจที่คงอยู่; หลีกเลี่ยงความลับ

## Heartbeat (ไม่บังคับ)

- `HEARTBEAT.md` สามารถเก็บเช็กลิสต์ขนาดเล็กสำหรับการรัน Heartbeat ได้; ควรให้สั้น

## ปรับแต่ง
