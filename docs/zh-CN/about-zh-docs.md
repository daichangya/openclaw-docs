# 关于中文文档 🇨🇳

欢迎使用 OpenClaw 中文社区文档！本文档由中文用户社区维护，旨在为中文用户提供更好的使用体验。

## 📖 文档说明

### 这是什么？

这是 [OpenClaw](https://github.com/openclaw/openclaw) 项目的中文翻译文档，提供：

- ✅ **完整的中文化**：所有核心文档均已翻译为简体中文
- ✅ **本土化示例**：包含微信、飞书等中国流行平台的配置指南
- ✅ **中文 FAQ**：针对中文用户的常见问题解答
- ✅ **社区支持**：欢迎中文用户提交改进建议

### 与官方文档的关系

| 项目 | 中文文档 | 官方文档 |
|------|---------|---------|
| 网址 | [clawcn.me](https://clawcn.me) | [docs.openclaw.ai](https://docs.openclaw.ai) |
| 语言 | 简体中文 | 英文 |
| 维护者 | 中文社区 | OpenClaw 官方团队 |
| 更新频率 | 定期同步官方更新 | 实时更新 |
| 特色内容 | 微信、飞书等本土化示例 | 国际平台为主 |

## 🎯 本土化特色

### 1. 中国平台支持

我们特别关注中国用户常用的平台：

- **微信 (WeChat)**: 详细的配置指南和常见问题
- **飞书 (Feishu)**: 企业级部署方案
- **QQ Bot**: QQ 机器人集成教程
- **钉钉**: 待添加（欢迎贡献）

### 2. 中文用户常见问题

针对中文用户的使用习惯，我们整理了：

- 国内网络环境下的部署方案
- 中文输入法和编码问题处理
- 时区和日期格式配置
- 国内云服务商部署指南（阿里云、腾讯云等）

### 3. 社区贡献

本文档是开源的，欢迎贡献：

- 📝 **翻译改进**：发现翻译错误？欢迎提交 PR
- 💡 **内容补充**：有独特的使用案例？欢迎分享
- 🔧 **配置示例**：有新的平台配置？欢迎添加
- ❓ **FAQ 整理**：遇到问题并解决？欢迎记录

## 🤝 如何贡献

### 报告问题

如果你发现：
- 翻译错误或不通顺的地方
- 过时的配置信息
- 缺失的重要功能说明

请在 [GitHub Issues](https://github.com/openclaw/openclaw/issues) 中报告，标签使用 `documentation` 和 `i18n-zh`。

### 提交翻译改进

1. Fork [openclaw/openclaw](https://github.com/openclaw/openclaw) 仓库
2. 修改 `docs/docs/zh-CN/` 目录下的文件
3. 提交 Pull Request
4. 等待审核合并

### 添加本土化内容

如果你想添加中国特有平台的支持：

1. 在 `docs/docs/zh-CN/channels/` 创建新文件
2. 参考其他渠道文档的格式
3. 提供完整的配置步骤和截图
4. 提交 Pull Request

## 📊 文档同步策略

我们采用以下策略保持与官方文档同步：

1. **自动同步**：通过 GitHub Actions 定期检测官方更新
2. **手动翻译**：社区志愿者翻译新增内容
3. **质量审查**：多人审核确保翻译质量
4. **版本标记**：标注每个页面的最后同步时间
