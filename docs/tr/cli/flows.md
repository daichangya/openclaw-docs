# `openclaw tasks flow`

Akış komutları, bağımsız bir `flows` komutu değil, `openclaw tasks` alt komutlarıdır.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Tam belgeler için bkz. [TaskFlow](/tr/automation/taskflow) ve [tasks CLI başvurusu](/tr/cli/tasks).
