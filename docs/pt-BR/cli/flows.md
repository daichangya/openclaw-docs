# `openclaw tasks flow`

Os comandos de fluxo são subcomandos de `openclaw tasks`, não um comando `flows` independente.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Para a documentação completa, consulte [Task Flow](/automation/taskflow) e a [referência da CLI de tasks](/cli/index#tasks).
