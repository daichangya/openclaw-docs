# TOOLS.md - Lokale Notizen

Skills definieren, _wie_ Tools funktionieren. Diese Datei ist für _Ihre_ Besonderheiten gedacht — also für die Dinge, die in Ihrer Einrichtung einzigartig sind.

## Was hier hineingehört

Dinge wie:

- Kameranamen und -standorte
- SSH-Hosts und -Aliase
- Bevorzugte Stimmen für TTS
- Namen von Lautsprechern/Räumen
- Gerätenamen
- Alles Umgebungsspezifische

## Beispiele

```markdown
### Cameras

- living-room → Hauptbereich, 180°-Weitwinkel
- front-door → Eingang, bewegungsgesteuert

### SSH

- home-server → 192.168.1.100, Benutzer: admin

### TTS

- Bevorzugte Stimme: "Nova" (warm, leicht britisch)
- Standardlautsprecher: Kitchen HomePod
```

## Warum getrennt?

Skills werden gemeinsam genutzt. Ihre Einrichtung gehört Ihnen. Wenn Sie beides trennen, können Sie Skills aktualisieren, ohne Ihre Notizen zu verlieren, und Skills teilen, ohne Ihre Infrastruktur preiszugeben.
