# `openclaw tasks flow`

Flow-Befehle sind Unterbefehle von `openclaw tasks`, kein eigenständiger `flows`-Befehl.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Die vollständige Dokumentation finden Sie unter [TaskFlow](/de/automation/taskflow) und in der [tasks-CLI-Referenz](/de/cli/tasks).
