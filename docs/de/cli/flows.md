# `openclaw tasks flow`

Flow-Befehle sind Unterbefehle von `openclaw tasks`, nicht ein eigenständiger `flows`-Befehl.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Die vollständige Dokumentation finden Sie unter [Task Flow](/automation/taskflow) und in der [tasks CLI-Referenz](/cli/index#tasks).
